## Übersicht
Ein einfaches Platformer Spiel mit 6 Levels

### Steuerung
| Taste       |                   |
|-------------|-------------------|
|**A**        |**Linksbewegung**  |
|**D**        |**Rechtsbewegung** |
|**[SPACE]**  |**Springen**       |
|R            |Level zurücksetzten|
|F1           |Level überspringen |
|F2           |Zurück zu Level 1  |
|F11          |Fullscreen         |

## Installation
  - ### Windows
    - Ein portables ZIP Archiv und eine Installer .exe steht auf der [Release Seite](https://github.com/fabi-deu/el-game/releases/latest) zur verfügung.
    - > Alternativ kann auch direkt vom Quellcode komplimiert werden.

  - ### MacOS
    - Bitte aus dem Quellcode kompilieren.
  
  - ### Linux
    - Bitte aus dem Quellcode kompilieren.


_Die vorgefertigten Release-Versionen sind nicht signiert._
_Deshalb könnten Windows (Defender / SmartScreen) oder andere Antivirus-Programme eine Warnung anzeigen._
